## The Three Sequential Bets

1. **Bet 1 (Month 0-12): The Value Gap** -- Structured .spec files deliver measurably better AI agent performance than unstructured context for any task, in any domain
2. **Bet 2 (Month 6-24): The Ecosystem** -- A zero-entity-core architecture with domain extensions reaches critical mass (30+ community extensions across 10+ domains; Terraform-exact model: core has zero domain knowledge, all vocabulary from extensions)
3. **Bet 3 (Month 18-48): The Standard** -- The Graph Protocol becomes the universal structured context format for AI agents across all domains -- software, compliance, design, data, infrastructure, and domains not yet imagined
